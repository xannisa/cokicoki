# cokicoki
3 different flavours

Task 1 is Strawberry_1

File health_check.sh provide basic health check on server. The checking method use ping, curl, and disk usage. Ping and Curl using 5 times checks. 

How to run :
From terminal, run the bash file using an argument.

Example :
bash health_check.sh localhost 443

If it doesnt run, change the file to executeable by using : 

chmod +x health_check.sh
